BOX MODEL

O Box Model é um conceito fundamental no CSS que descreve a estrutura de um elemento HTML como uma caixa retangular. Ele é composto por quatro áreas principais: conteúdo, preenchimento (padding), borda (border) e margem (margin). O conteúdo é a área onde o texto ou os elementos filhos são exibidos. O preenchimento é o espaço entre o conteúdo e a borda, enquanto a borda é a linha que envolve o elemento. A margem é o espaço entre a borda do elemento e os elementos adjacentes. Compreender o Box Model é essencial para controlar o layout e o espaçamento dos elementos em uma página web.



CSS GRID

O CSS Grid é um sistema de layout bidimensional que permite criar designs complexos e responsivos de forma fácil e eficiente. Ele divide a página em linhas e colunas, permitindo que os elementos sejam posicionados em qualquer lugar dentro dessa grade. Com o CSS Grid, é possível criar layouts flexíveis e adaptáveis, facilitando a organização do conteúdo em diferentes dispositivos e tamanhos de tela.



FLEXBOX

O Flexbox é um modelo de layout unidimensional que facilita a distribuição de espaço entre os itens em um contêiner, mesmo quando seu tamanho é desconhecido ou dinâmico. Ele permite alinhar e distribuir os itens de forma eficiente, seja horizontalmente ou verticalmente. Com o Flexbox, é possível criar layouts flexíveis e responsivos, ajustando automaticamente o tamanho dos itens para se adequar ao espaço disponível.



DISPLAYS

O display é uma propriedade CSS que define como um elemento HTML é exibido na página. Existem vários valores para a propriedade display, cada um com um comportamento diferente. Alguns dos valores mais comuns incluem block, inline, inline-block, flex e grid. O valor block faz com que o elemento ocupe toda a largura disponível e comece em uma nova linha, enquanto o valor inline permite que o elemento seja exibido na mesma linha que outros elementos. O valor inline-block combina as características de block e inline, permitindo que o elemento seja exibido na mesma linha, mas ainda ocupando apenas o espaço necessário. O valor flex é usado para criar layouts flexíveis, enquanto o valor grid é usado para criar layouts em grade.

CSS GRID

O valor grid para a propriedade display é usado para criar layouts em grade. Ele permite que os elementos sejam organizados em linhas e colunas, facilitando a criação de designs complexos e responsivos. Com display: grid, é possível definir áreas específicas para cada elemento, controlar o espaçamento entre eles e criar layouts flexíveis que se adaptam a diferentes tamanhos de tela.

FLEX

O valor flex para a propriedade display é usado para criar layouts flexíveis. Ele permite que os elementos sejam organizados em uma única linha ou coluna, facilitando a distribuição de espaço entre os itens. Com display: flex, é possível alinhar os itens horizontalmente ou verticalmente, controlar o tamanho dos itens e criar layouts responsivos que se adaptam a diferentes tamanhos de tela.

Um Grid-Container CSS é o elemento HTML pai que tem a propriedade display: grid ou display: inline-grid aplicada, tornando-se o contêiner principal de um sistema de layout bidimensional (linhas e colunas). Ele organiza os seus filhos diretos (grid items) em uma estrutura de grade precisa, ideal para layouts complexos e responsivos.

Características de um Grid-Container CSS:

  • Definição: Cria um novo contexto de formatação de grade para seus elementos filhos.
  • Colunas e Linhas: Define-se o número e tamanho com grid-template-columns e grid-template-rows.
  • Espaçamento: Usa-se gap (ou grid-gap) para definir o espaço entre linhas e colunas.
  • Áreas Nomeadas: grid-template-areas permite nomear regiões, facilitando o posicionamento de itens.
  • Alinhamento: Controla o alinhamento de todos os itens com justify-items (horizontal) e align-items (vertical).

GRID LAYOUT

O Grid Layout (ou CSS Grid) é um sistema bidimensional de layout CSS que organiza elementos na web usando linhas e colunas, permitindo criar designs complexos e responsivos com eficiência. Ele é "bidimensional" porque manipula linhas e colunas simultaneamente, facilitando o alinhamento e a estruturação de layouts sem os truques do passado.

Principais características e conceitos:

  • Bidimensional: Controle total sobre linhas (rows) e colunas (columns) ao mesmo tempo.
  • Contêiner e Itens: O elemento pai é definido como display: grid, tornando os elementos filhos, ou itens, posicionáveis na grade.
  • Estrutura: Utiliza grid-template-columns e grid-template-rows para definir o tamanho da malha.
  • Espaçamento: Propriedades como gap (ou grid-gap) definem facilmente o espaço entre colunas e linhas.
  • AÁreas Nominais: É possível criar áreas de layout nomeadas (grid-template-areas) para organizar cabeçalhos, rodapés e barras laterais de forma intuitiva.
  • Unidades de Medida: Introduz a unidade $fr$ (fração) para criar tamanhos proporcionais no grid.
