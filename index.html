2. BOX MODEL

O Box Model é um conceito fundamental no CSS que descreve a estrutura de um elemento HTML como uma caixa retangular. Ele é composto por quatro áreas principais:

  • Conteúdo: Onde o texto e as imagens aparecem.
  • Padding (Preenchimento): Espaço entre o conteúdo e a borda.
  • Border (Borda): A linha que envolve o preenchimento.
  • Margin (Margem): Espaço por fora da borda, separando outros elementos.

3. PROPRIEDADE DISPLAY

BLOCK

Elementos de bloco ocupam toda a largura disponível e começam em uma nova linha.

Exemplo de elemento Block

INLINE

Elementos inline ocupam apenas o espaço necessário para o seu conteúdo e não aceitam largura/altura fixa.

INLINE-BLOCK

Combina os dois: os elementos ficam lado a lado (como inline), mas permitem definir largura, altura e margens (como block). Exemplos de tags comuns: <span>, <a>, <strong>, <img>.

4. CSS GRID

O CSS Grid é um sistema de layout bidimensional (linhas e colunas). Ele permite criar designs complexos e responsivos de forma intuitiva.

Principais conceitos:

  • Bidimensional: Controle total sobre linhas (rows) e colunas (columns) ao mesmo tempo.
  • Contêiner e Itens: O elemento pai vira um grid, e os filhos viram itens da grade.
  • Unidade fr: Representa uma fração do espaço livre no contêiner.
  • Gap: Define o espaço entre as células da grade.

5. FLEXBOX

Diferente do Grid, o Flexbox é unidimensional (foca ou em linha ou em coluna por vez). É excelente para alinhar itens e distribuir espaço de forma dinâmica dentro de um contêiner.
