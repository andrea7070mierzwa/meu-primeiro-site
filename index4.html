SELETORES CSS

Os seletores CSS são usados para selecionar os elementos HTML que queremos estilizar. Eles permitem que você aplique estilos específicos a elementos específicos em sua página da web.

Seletores Básicos e de Combinadores:

  • Universal (*): Seleciona todos os elementos.
  • Tipo/Tag (p): Seleciona todos os elementos.
  • Classe (.classe): Seleciona todos os elementos com class="classe".
  • ID (#id): Seleciona um único elemento com id="id".
  • Atributo ([atrib], [atrib=valor]): Seleciona com base na presença ou valor de um atributo.

DESAFIO: FORMULÁRIO DE CONTATO


Crie um formulário de contato usando HTML e CSS. O formulário deve incluir os seguintes campos:


  • Nome (campo de texto)
  • Email (campo de email)
  • Idade (campo numérico)
  • Matéria (campo de seleção com opções como Português, Matemática, Biologia, etc.)
  • Escolaridade (campo de seleção com opções como Licenciatura Cursando, Licenciatura Concluída, Graduação Cursando, etc.)
  • Disponibilidade (checkboxes para opções como Integral, Matutino, Vespertino, Noturno)
  • Mensagem (campo de texto para o usuário escrever uma mensagem)
